# Login — Manual Test Cases

| Test Case ID | Title | Feature | Pre-conditions | Steps | Expected Result | Test Type | Priority |
|---|---|---|---|---|---|---|---|
| TC_LOGIN_001 | Successful login with valid credentials | Login | 1. User is on the login page (`/users/sign_in`) 2. User has a valid registered account | 1. Navigate to `https://tipton.sandbox.usemast.com/users/sign_in` 2. Verify the heading reads "Welcome back, let's sign in!" 3. Enter a valid email in the "Enter email address..." field 4. Enter the correct password in the "Enter password..." field 5. Click the "Sign in" button | User is redirected to `https://tipton.sandbox.usemast.com/`. Navigation bar is visible with Cases, Tasks, Dashboard, Product Engine, Policy Engine. Notifications and Account buttons are visible. No error messages displayed. | Positive | High |
| TC_LOGIN_002 | Login with empty email and empty password | Login | 1. User is on the login page (`/users/sign_in`) 2. Both fields are empty | 1. Click the "Sign in" button | User remains on `/users/sign_in`. Error message displayed: **"Invalid Email or password."** No redirect occurs. | Negative | High |

| TC_LOGIN_012 | Successful logout from the application | Login — Logout | 1. User is logged in and on any page of the application | 1. Click the "Account" button in the top-right navigation 2.
